 Accept a new connection.
 Принять новое соединение.

 * Эта функция используется для принятия нового соединения от узла в
 * указанный сокет. Вызов функции будет блокироваться до тех пор, пока новое соединение не будет

 * успешно принято или не произойдет ошибка.

 *
 * @param peer Сокет, в который будет принято новое соединение.

 *

 * @param ec Указывает, какая ошибка произошла, если таковая имеется.

 *

 * @par Пример

 * @code
 * boost::asio::ip::tcp::acceptor acceptor(my_context);

 * ...
 * boost::asio::ip::tcp::socket socket(my_context);

 * boost::system::error_code ec;

 * acceptor.accept(socket, ec);

 * if (ec)

 * {
 * // Произошла ошибка.

 * }

 * @endcode

 /// Start an asynchronous accept.
Начать асинхронное принятие.

* Эта функция используется для асинхронного приема нового соединения в
* сокет, а также для получения конечной точки удаленного узла. Она является
* инициирующей функцией для @ref asynchronous_operation и всегда возвращает
* немедленно.

*

* @param peer Сокет, в который будет принято новое соединение.

* Право собственности на объект узла сохраняется за вызывающей стороной, которая должна
* гарантировать его действительность до вызова обработчика завершения.

*

* @param token Токен завершения @ref completion_token, который будет использоваться для создания
* обработчика завершения, который будет вызван после завершения приема.

* Возможные токены завершения включают @ref use_future, @ref use_awaitable,
* @ref yield_context или объект функции с правильной сигнатурой завершения.
* Сигнатура функции обработчика завершения должна быть:

* @code void handler(
* const boost::system::error_code& error // Результат операции.

* ); @endcode
* Независимо от того, завершится ли асинхронная операция немедленно или нет,
* обработчик завершения не будет вызываться из этой функции.

* При немедленном завершении вызов обработчика будет выполнен
* способом, эквивалентным использованию boost::asio::post().

*
* @par Подпись завершения

* @code void(boost::system::error_code) @endcode

*
* @par Пример

* @code
* void accept_handler(const boost::system::error_code& error)

* {

* if (!error)

* {
* // Принятие прошло успешно.

* }

* }
*
* ...

*
* * boost::asio::ip::tcp::acceptor acceptor(my_context);

* ...

* boost::asio::ip::tcp::socket socket(my_context);

* acceptor.async_accept(socket, accept_handler);

* @endcode
